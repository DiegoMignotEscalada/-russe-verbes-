ASPECT : IMP. VS PF.
L'imperfectif = action en cours, répétée, habituelle.
Le perfectif = action accomplie, ponctuelle, avec résultat.

Я читал книгу — j'étais en train de lire
Я прочитал книгу — j'ai fini de lire le livre
PRÉSENT TYPE 1 (-ать/-еть)
я -ю / ты -ешь / он -ет / мы -ем / вы -ете / они -ют

Ex: читать → читаю, читаешь, читает...
PRÉSENT TYPE 2 (-ить/-еть)
я -ю(-у) / ты -ишь / он -ит / мы -им / вы -ите / они -ят(-ат)

Ex: говорить → говорю, говоришь, говорит...
PASSÉ
Infinitif sans -ть + genre : -л / -ла / -ло / -ли
Ex: читать → читал, читала, читало, читали
IMPÉRATIF
Base présent (3pl. - ют/ат) :
→ voyelle finale : + й / йте
→ consonne (accent sur rad.) : + и / ите
→ consonne (accent avant) : + ь / ьте
ALTERNANCES CONSONANTIQUES (1ère pers.)
б→бл · в→вл · д→ж · з→ж · с→ш · т→ч · ск→щ
